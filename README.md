# Fuel_Price_Watchdog_Pipeline
## Overview
This project tracks daily fuel prices, detects changes, and generates trend-based insights.

## Features
- Daily price change detection (Increase/Decrease)
- 7-day and 30-day trend analysis
- SQLite database storage
- Automated reporting system

## Technologies Used
- Python
- Pandas
- SQLite

## How to Run
1. Run main pipeline:
   python 1_Fuel_Project.py

2. View stored data:
   python 2_view_data.py

## Output
- Daily price report
- Weekly trend summary

## Dataset
Fuel price data for multiple cities over 45 days
